# Syntecxhub_Feasto_Mobile-Food-Delivery-App

## Project Overview
Feasto is a modern food delivery application UI/UX design that focuses on providing a clean, intuitive, and scalable user experience for food ordering platforms.

## Objective
To design a user-friendly food delivery interface with a strong design system that simplifies browsing, ordering, and tracking food deliveries.

## Target Users
- Students
- Working professionals
- Urban food delivery users

## Tools Used
- Figma

## UX Focus
- Simple navigation
- Clear CTAs
- Consistent UI patterns
- Easy checkout flow

## Key Features
- Food categories & restaurant listing
- Cart & checkout
- Order tracking
- User profile & account screens
- Design system & reusable components
